# Selenium__Framework_from_Scratch
Selenium Test Automation Framework (Java + Maven + TestNG)

This is a lightweight, scalable test automation framework built using Selenium WebDriver, TestNG, and Maven.
It supports modular test design, HTML & XML reporting, screenshots on failure, and CI integration with Azure DevOps.

Tech Stack:

Language: Java
Automation: Selenium WebDriver
Testing Framework: TestNG
Build Tool: Maven
Design Pattern: Page Object Model (POM)
Reports: TestNG HTML, Surefire JUnit XML
CI/CD: Azure DevOps
Screenshot: File + Base64 on test failure

Key Features:

Page Object Model (POM)
Centralized driver management
Custom TestNG Listener
Screenshots on failure
HTML + XML reports
Optimized for CI/CD
Clean, maintainable structure
